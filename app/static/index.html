How to use NWChem Studio

  

    
1. Create a job

    
Click + New job. Paste your molecule as XYZ coordinates — one atom
       per line: element x y z in ångströms (the 3D
       preview updates as you type). Choose the charge, spin multiplicity,
       level of theory (DFT, Hartree–Fock or MP2), functional, basis set and
       the task:

    
    
      
  • Single-point energy — energy at the given geometry.
    • 
      
  • Geometry optimization — relax the structure to a minimum.
    • 
      
  • Optimization + frequencies — optimize, then compute vibrations
          (recommended: confirms you found a true minimum).
    • 
    

    
The generated NWChem input file is shown live; after creating the job you
       can hand-edit it in the Input tab before running.


    
2. Run it

    
Press ▶ Run. The Output log tab streams NWChem's raw output
       live; you can ■ Cancel a job at any time. The badge in the sidebar
       shows whether a compute backend (Docker or built-in NWChem) is available.


    
3. Explore the results

    
    
      
  • 3D structure — rotate (drag), zoom (scroll). Switch between
          ball-and-stick, spacefill and wireframe; toggle atom labels.
    • 
      
  • Trajectory — for optimizations, use the frame slider or
          ▶ Play to watch the geometry relax.
    • 
      
  • Energy chart — energy at each optimization step; it should
          decrease and flatten out when converged.
    • 
      
  • Frequencies — click any value to animate that vibrational mode.
          Red (negative) values are imaginary frequencies: the structure is a
          saddle point, not a minimum.
    • 
      
  • Summary cards — final energy (in Hartree), convergence status,
          dipole moment and imaginary-frequency count.
    • 
    


    
Visualize existing files

    
Open output file… loads any NWChem .out file (or a
       plain .xyz) without running anything — useful for results
       computed elsewhere, e.g. on a cluster.


    
Tips

    
    
      
  • Energies are in Hartree (1 Ha = 27.211 eV = 627.5 kcal/mol).
    • 
      
  • Start small: sto-3g or 3-21g for quick tests, 6-31g* or def2-svp for
          production geometries.
    • 
      
  • If a job fails, read the end of the Output log — NWChem prints
          the reason there. If it runs out of memory, add a line like
          memory 400 mb to the input.
    • 
      
  • ⚙ Settings lets you change the NWChem Docker image.
